# northstar-updater

Lightweight update workflow sandbox focused on reliability and cleanup discipline.

## Why this exists
I built this to reduce release chaos in small projects:
- fewer “temporary” doc leftovers
- cleaner update notes
- better rollback hygiene
- repeatable pre-release checks

## Current scope
- basic update flow validation
- docs/process cleanup
- CI sanity checks
